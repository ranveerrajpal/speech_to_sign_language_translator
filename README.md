Speech to Sign Language Translator

A complete accessibility-focused web application that converts spoken language into sign language animations/videos in real time. This project was developed to bridge the communication gap between hearing individuals and the deaf or hard-of-hearing community.

🚀 Project Overview

The Speech to Sign Language Translator captures speech input, converts it into text using Vosk Speech Recognition, processes the transcript, and displays corresponding sign language gestures through video clips or animations.

The system is designed to provide fast, efficient, and user-friendly communication assistance in schools, workplaces, hospitals, and daily conversations.

✨ Features
🎤 Real-time Speech Input
📝 Accurate Offline Speech Recognition using Vosk
🤟 Text to Sign Language Conversion
🎥 Auto-playing Sign Language Video Output
🌐 Web-Based Interface
⚡ Fast and Lightweight Processing
🔒 Offline Capability (No internet required for transcription)
🛠️ Technologies Used
-Python
-FastAPI (depending on your backend)
-Vosk API for speech-to-text transcription
-HTML / CSS / JavaScript
-FFmpeg for audio processing
-Sign Language Video Dataset

⚙️ How It Works
-User speaks through microphone.
-Audio is captured by the system.
-Vosk converts speech into text transcript.
-Transcript is broken into words/phrases.
-Matching sign language videos are played sequentially.
-User receives visual sign language output.
